# Introduction to Machine Learning with Python 

This repository contains my reproduction and study notes based on the book *"Introduction to Machine Learning with Python: A Guide for Data Scientists"* by Andreas C. Müller and Sarah Guido.

## Project Overview
The goal of this repository is to build a practical foundation in machine learning by implementing the core concepts and algorithms discussed in the book. Each folder contains a Jupyter Notebook corresponding to a specific chapter, complete with:
- Reproductions of the book's Python code (utilizing `scikit-learn`, `numpy`, `pandas`, and `matplotlib`).
- Theoretical explanations of the underlying machine learning models and mathematical concepts.
- A concise summary of the chapter's key takeaways.

## Repository Contents

* **[Chapter 1: Introduction](./Chapter_1_Introduction)** * *Overview:* Basic concepts, setting up the environment, and a first application (Iris dataset).
* **[Chapter 2: Supervised Learning](./Chapter_2_Supervised_Learning)**
    * *Overview:* Classification and regression, generalization, overfitting, and core algorithms like k-Nearest Neighbors, Linear Models, Naive Bayes, Decision Trees, Ensembles, SVMs, and Neural Networks.
* **[Chapter 3: Unsupervised Learning and Preprocessing](./Chapter_3_Unsupervised_Learning)**
    * *Overview:* Data transformations, scaling, dimensionality reduction (PCA, NMF, t-SNE), and clustering algorithms (k-Means, Agglomerative, DBSCAN).
